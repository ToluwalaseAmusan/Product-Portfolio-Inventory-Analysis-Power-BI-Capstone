Product Portfolio & Inventory Analysis | Power BI Capstone
Project Overview This project involved developing an executive-level Power BI dashboard to analyze an organization’s $128.83M product portfolio. The analysis focuses on inventory valuation, category distribution, and lifecycle risk assessment to support data-driven decision-making.

Key Metrics
Total Inventory Value: $128.83M

Product Count: 10,000 products

Total Stock Units: 506,000 units

Average Product Rating: 3.00

Strategic Insights
Balanced Portfolio: Inventory value is evenly distributed (~$43M each) across Electronics, Home Appliances, and Clothing.

High-Value Concentration: Specific products like Smartphones, Monitors, and Laptops drive the highest value, individually contributing between $31M and $33M.

Operational Risk: Identified a critical risk with 7,000 expired products and 3,000 products expiring soon, indicating potential value erosion.

Technical Skills Demonstrated
Data Visualization: Created interactive dashboards in Power BI to track trends over time.

Risk Modeling: Developed a lifecycle analysis to identify "Value at Risk" based on product aging and expiration.

Business Intelligence: Translated raw inventory data into actionable recommendations for executive management.

Recommendations Provided
Risk Controls: Implement proactive monitoring for expiring stock to prevent write-offs.

Portfolio Optimization: Review low-performing products for rationalization to improve capital allocation.

High-Value Protection: Apply tighter stock controls on top-tier inventory assets.
